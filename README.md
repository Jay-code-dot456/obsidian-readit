# 悦读 Readit

一款 Obsidian 沉浸式阅读插件:护眼配色、可调排版、铺满窗口、自动平滑滚动、阅读进度记忆,并支持一键打"入库 / 已读"标签。全平台可用(桌面 + 安卓 / iOS)。

## ✨ 功能特性

- **护眼配色**:羊皮纸(米黄)、纸白、豆沙绿、暗夜四套预设,仅作用于中间阅读区,不影响侧边栏。
- **排版可调**:字体、字重、字号、行距、行宽;默认黑体优先的跨平台字体回退链,手机上更清晰。
- **沉浸阅读**:一键铺满 Obsidian 窗口(隐藏标签栏/状态栏/图标栏/标题栏),可选折叠侧边栏、系统级全屏。
- **自动平滑滚动**:基于逐帧渲染的亚像素级平滑滚动,速度可在阅读时即时调节(慢 / 快),视线固定即可阅读。
- **阅读进度记忆**:阅读进度写入文档头部 `readit_progress` 字段,下次打开自动跳回上次位置。
- **标签速记**:一键为当前文档打 / 取消 `KBase`(入库)与 `readed`(已读)标签。
- **多入口、全平台**:左侧图标、命令面板、笔记"⋮ 更多选项"菜单均可进入;移动端通过浮动按钮操作,无需键盘。

## 🚀 安装

### 手动安装

1. 从 [Releases](../../releases) 下载或在本仓库获取 `main.js`、`manifest.json`、`styles.css`;
2. 放入你的库目录:`<你的库>/.obsidian/plugins/readit/`;
3. 在 Obsidian 中 `设置 → 第三方插件` 关闭安全模式,刷新并启用「悦读 Readit」。

### 安卓 / iOS

把上述三个文件放进手机上库的 `.obsidian/plugins/readit/`(需使用可显示隐藏文件的文件管理器),启用即可。

## 📖 使用

- **进入沉浸阅读**:点左侧 📖 图标 / 命令面板「切换沉浸阅读」/ 笔记右上角 ⋮ 菜单「进入沉浸阅读」。
- **退出**:按 `ESC`(桌面)或点右下角 ✕ 按钮(全平台)。
- **浮动按钮**(沉浸阅读中,右下角):
  - `A− / A+`:调字号
  - `▶ / ⏸`:开始 / 暂停自动滚动;滚动时浮现「慢 / 快」即时调速
  - `入库 / 已读`:切换标签(开启时高亮)
  - `✕`:退出

所有外观与行为均可在 `设置 → 悦读 · 沉浸阅读设置` 中调整。

## 🛠 开发

```bash
npm install
npm run dev    # 监听编译
npm run build  # 生产构建
```

源码主文件为 `main.ts`,使用 esbuild 打包为 `main.js`。
